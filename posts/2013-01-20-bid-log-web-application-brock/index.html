Bid Log Application (Brock)


The client wanted an internal Purchase Order System to keep track of what the company was buying. I worked with the Plant Manager and the Accounts Receivable person to create a custom Purchase Order System that fit thier business. This site was developed using ASP.NET, C#, JQuery, HTML, and CSS.

This Bid Log Web Application was built for and used by all Brock Companies (Brock in all 5 regions, Brock's Union side, Brock Canada, and Brock Latin America). There were over 130 unique estimators who used it at a given time. This Web Application also had reporting built into the website for each company to pull whenever they wanted. Each company used the Bid Log to see what was work was coming down the line, internal communication, and budget forecasting. Upper Management used the Bid Log reports for forecasting future revenues, reporting to the bank / owners / board, and to manage the business.

The Bid Log Web Application emailed estimators a copy of the bid they entered or updated. This Web Application was build on the ASP.NET platform using ASP.NET, C#, Razor, Microsoft SQL Server, JQuery, HTML, and CCS. Extra form validation was coded for this Web Application via Jquery to help users put in correct data. Built for functionality, user interface, and ease of use.
