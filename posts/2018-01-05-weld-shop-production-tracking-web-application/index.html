Weld Shop Production, Quality Control, and Material Tracking Web Application


This client wanted a better way to track projects than using paper and calling shop foreman for updates. With the help of a Production Manager, I was able to come up with this weld / production tracker web application. This application was developed using ASP.NET MVC, C#, JQuery, HTML, and CSS. The user has the ability to upload pipe information from a drafting program and into this web application.

The Web App Design and friendly User Interface was designed by myself with additional ideas from Project Managers and the Head of IT. This application was built for multiple people and departments including: Engineering, Project Managers, Shop Foreman, Quality Control (QAQC), Receiving, and the Plant Manager.

This application helps these users and departments track their work with less interuptions and less paperwork. This is the most in depth Web Application I have built to date.
