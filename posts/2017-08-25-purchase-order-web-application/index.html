Purchase Order Web Application


The client wanted an internal Purchase Order System to keep track of what the company was buying. I worked with the Plant Manager and the Accounts Receivable person to create a custom Purchase Order System that fit thier business. This site was developed using ASP.NET, C#, JQuery, HTML, and CSS.

The Web App Design and friendly User Interface were designed by myself. Functionality was a team effort between the Plant Manager, Accounts Receivable person, and myself.
