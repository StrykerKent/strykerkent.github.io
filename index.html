Bid Log Application 1 (Brock)

Bid Log Web Application

Bid Log (Brock) - Web Application

This Bid Log Web Application was built for and used by all Brock Companies (Brock in all 5 regions, Brock's Union side, Brock Canada, and Brock Latin America). There were over 130 unique estimators who used it at a given time. This Web Application also had reporting built into the website for each company to pull whenever they wanted. Each company used the Bid Log to see what was work was coming down the line, internal communication, and budget forecasting. Upper Management used the Bid Log reports for forecasting future revenues, reporting to the bank / owners / board, and to manage the business.

The Bid Log Web Application emailed estimators a copy of the bid they entered or updated. This Web Application was build on the ASP.Net platform using ASP.NET, C#, Razor, Microsoft SQL Server, JQuery, HTML5, and CCS3. Extra form validation was coded for this Web Application via Jquery to help users put in correct data. Built for functionality, user interface, and ease of use.

Client: The Brock Group
Website: (An Intranet employees only have access to.)
Date: 1/20/2013
Service: Web Development, Database Application, User Interface, Ease of Use
