Equipment Transfer Web Application


Live Demo

This was a proof of concept for a company that was having trouble keeping up with their equipment and the location it was in. They wanted a Mobile Web Application they could use on their phone and desktops for requesting equipment transfers.

The app grabs the user's geo location and allows the user to upload a picture of the equipment. It also has fields for the user to fill out about the equipment to complete the request. This was developed using PHP, MySQL, jQuery, HTML, and CSS.
The design and user experience were designed by myself.


Client: Echo Maintenance
Demo Website: https://sksapp.000webhostapp.com/
Service: Mobile Web Development, User Experience, PHP, MySQL, LAMP Stack
